# Jersey Lab - Custom Football Jersey Designer

## Project Overview

Jersey Lab is a custom football jersey designer application that allows users to design their perfect football jersey with custom names and numbers. Choose from Premier League, La Liga, Bundesliga, Serie A, and Ligue 1 clubs.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd jersey-vision-studio

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

## Technologies Used

This project is built with:

- **Vite** - Fast build tool and dev server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **Three.js** - 3D graphics and jersey preview
- **shadcn-ui** - UI component library
- **Tailwind CSS** - Utility-first CSS framework

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run build:dev` - Build for development
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Features

- 3D jersey preview with real-time customization
- Multiple league support (Premier League, La Liga, Bundesliga, Serie A, Ligue 1)
- Custom name and number personalization
- Interactive design interface
- Responsive design for all devices

## Deployment

Build the project for production:

```sh
npm run build
```

The built files will be in the `dist` folder, ready to be deployed to any static hosting service.
